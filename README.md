## :shield: Standarts
Container's behavior is closed to C++ STL. All of objects can be used with STL algorithms. More information \
about interface or realization you can find in documentation.

#

## ✔️: Advantages:

+ non-standart container
+ simple code (for beginers)
+ compatible with STL algorithms and other standart libraries
+ custom exceptions
+ quick compiling and quick runtime using
+ huge system of unit-tests

#

## 🚀 Getting started

1. :arrow_down: Clone the repository:
```bash
git clone https://github.com/Shangin-Leonid/cyclic_list
cd ./cyclic_list
```
2. ➕➕ Use the `cyclic_list` just by including it:
```c++
#include <path_to_repo>/cyclic_list/cyclic_list.hpp
```


3. :gear: To check and benchmark compiling by `makefile`:
```bash
cd ./unit-tests
make do_compiling_test
```
*This will generate a report (read console input to have more details).*

4. :gear: To run unit-tests by `makefile`:
```bash
make do_test
```
*This will generate a report (read console input to have more details).*

5. :wastebasket: To clean all generated folders and files by `makefile`:
```bash
make clean_all
```
